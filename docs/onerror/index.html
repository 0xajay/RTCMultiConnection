    • onerror | RTCMultiConnection

    This event is fired only if WebRTC data channel fails to connect.

    Usage

    connection.onerror = function(event) {
    var remoteUserId = event.userid;
    var remoteUserFullName = event.extra.fullName;

    alert('unable to open data connection between you and ' + remoteUserFullName);
};

    Description

    parameterdescription
    userid unable to open data connection between you and this user-id 
    connection.onerror = function(event) {
    var remoteUserId = event.userid;
};
    extra extra information 
    connection.onerror = function(event) {
    var remoteUserFullName = event.extra.fullName;
};
