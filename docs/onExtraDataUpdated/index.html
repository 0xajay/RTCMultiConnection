RTCMultiConnection.onExtraDataUpdated | "event"

HOME © Muaz Khan . @WebRTCWeb . Github . Latest issues . What's New?

"connection.onExtraDataUpdated" event is fired for all users as soon as someone updates his extra-data.

"onExtraDataUpdated" event requires RTCMultiConnection >= v3.

An event object is passed over "onExtraDataUpdated": 

connection.onExtraDataUpdated = function(event) {
    var extra = event.extra;
    var targetUserId = event.userid;
};
