"autoCloseEntireSession" boolean

"autoCloseEntireSession" can be used to force closing the entire session when initiator leaves.

By default; autoCloseEntireSession is "false". It means that when initiator leaves; first participant will be given the role of conference owner.

How to use?

connection.autoCloseEntireSession = true;

leaveOnPageUnload == false?

Make sure that leaveOnPageUnload is true, otherwise autoCloseEntireSession will fail to work. 

connection.leaveOnPageUnload = true;
connection.autoCloseEntireSession = true;
